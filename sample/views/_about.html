About QuickRoute

QuickRoute is a simple and light-weight JavaScript routing framework for SPAs

This is a simple, light-weight JavaScript framework to create uri route based SPAs. With this framework you can create a JavaScript based Single Page Application very easliy, quickly and without the need to comprehend a complex library. This framework will add uri-hash based routing to the application.

You can look at the source code at GitHub.

To learn how to use it, head over to the training page.
